# Political-Quiz: https://13scales.github.io

13 Scales places you on thirteen political axes, as well as three compasses: economic/class issues, social issues, and foreign policy. It also provides an overall ideology and stance. You will respond to 50 statements, using a slider to indicate how strongly you agree or disagree with each one
